Event Overview

SMCC-CCIS Week 2025 is a five-day event that celebrates technology, innovation, creativity, and teamwork among students.

Students will participate in coding competitions, cybersecurity seminars, robotics demonstrations, project exhibitions, and exciting technology activities.

Event Highlights

  • Hackathon
  • Robotics Demo 🤖
  • Cybersecurity Seminar 🔒
  • Project Showcase
  • Awarding Ceremony 🏆
